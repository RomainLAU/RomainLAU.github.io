Who am I

Hi ! I'm Romain LAURENT, aka Le Roi Adibou, a french student in web development at Epitech Digital in Paris. I started coding 3 years ago, in 2019, by making little programs in Python. In 2020 I discovered the web development -- with HTML, CSS and PHP -- and at this moment, I knew that was what I wanted to do later. I love to start new projects where I don't know how I will do it, but I just know I'll do it. I like to work during night to fully be in peace, when nobody can disturb me, and there only are my project and me. I have always loved colors (as you can see on my hair), and that is why I had an art class in high school. And this, is why, even if I'm a developer, I'm really interested in design and UX/UI.

My Projects

Escape Game Show the project

During the first lockdown in 2020, I learnt how to use Unreal Engine 4 and I built an escape game based on what we had learnt in class. (Logic gates, Binary, Directories system in Windows, Text Hashing) This project took me around 150 hours of hard work and made me push my technical limits.

Unreal Engine 4

Random Shapes and Colors Show the project

In highschool I had an art class, and for a project I decided to use Python (with turtle module) to build a shape generator, you can decide of the shape (square, circle, triangle or parallelepiped), and the number of turns (the number of shapes drawn), and it will choose randomly a position, a color and a size for each shape.

Python

Karaoke Show the project

I developed a karaoke using JavaScript which displays lyrics while the music is playing. Using a file containing lyrics and timings to show them. I used Regex to sort timings and lyrics of the text file, and using AJAX method to display lyrics on the page.
